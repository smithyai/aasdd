## Document domain

Types representing the input document and the links it contains.

### Document

A piece of content that may contain links.

#### Properties

| Name | Type | Description |
| --- | --- | --- |
| `content` | text | The raw document content. |

### Link

A URL found within a document.

#### Properties

| Name | Type | Description |
| --- | --- | --- |
| `url` | text | The URL string. |
